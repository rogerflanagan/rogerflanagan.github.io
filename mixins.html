MIXINS

A mixin, unlike a class, represents a behavior. This is a group of styles that represents how objects work, not what they are. A mixin usually doesn't exist by itself; instead, it is included in classes that want to behave in a certain fashion. We have a few @extend's peppered throughout Zephyr. These are in the process of being removed. Please do not use extends at all!

fill-parent

@mixin fill-parent() {...}
  • Forces the element to take up the full width of parent
  • No params for this mixin

Example

.element {
    @include fill-parent;
}
.element {
    box-sizing: border-box;
    width: 100%;
}

absolute-center

@mixin absolute-center() {...}
  • Forces the element to be vertically and horizontally centered in its parent container
  • No params for this mixin

Example

.element {
    @include absolute-center;
}
.element {
    left: 50%;
    position: absolute;
    top: 50%;
    transform: translate3d(-50%,-50%,0);
}

horizontally-center

@mixin horizontally-center() {...}
  • Forces the element to be horizontally centered in its parent container
  • No params for this mixin

Example

.element {
    @include horizontally-center;
}
.element {
    left: 50%;
    position: absolute;
    transform: translate3d(-50%,0,0);
}

vertically-center

@mixin vertically-center() {...}
  • Forces the element to be vertically centered in its parent container
  • No params for this mixin

Example

.element {
    @include vertically-center;
}
.element {
    position: absolute;
    top: 50%;
    transform: translate3d(0,-50%,0);
}

auto-margin

@mixin auto-margin() {...}
  • Quick way to center an element without defining top and bottom margins
  • No params for this mixin

Example

.element {
    @include auto-margin;
}
.element {
    margin-left: auto;
    margin-right: auto;
}

z-table-layout

@mixin z-table-layout() {...}
  • Layout with equal width columns
  • No params for this mixin

Example

.element {
    @include z-table-layout;
}
.element {
    display: table;
    table-layout: fixed;
    width: 100%;
}

.element > * {
    display: table-cell;
}

pseudo-helper

@mixin pseudo-helper() {...}
  • Shortcut helper for building pseudo elements
  • No params for this mixin

Example

.element:before {
    @include pseudo-helper;
}
.element:before {
    content: '';
    position: absolute;
}

clearfix

@mixin clearfix() {...}
  • Reset Block Formatting Context on floated elements
  • No params for this mixin

Example

.element {
    @include clearfix;
}
.element:after {
    clear: both;
    content: '.';
    display: block;
    height: 0;
    visibility: hidden;
}

z-truncate

@mixin z-truncate() {...}
  • Shortcut for truncating a line of text
  • No params for this mixin

Example

.element {
    @include z-truncate;
}
.element {
    overflow: hidden;
    -ms-text-overflow: hidden;
    text-overflow: hidden;
    white-space: nowrap;
    word-wrap: normal; // IE9 fix
}

flex-img

@mixin flex-img() {...}
  • Shortcut for making images responsive within a container
  • No params for this mixin

Example

.element {
    @include flex-img;
}
.element {
    display: block;
    max-width: 100%;
}

background-cover

@mixin background-cover($imgUrl) {...}
  • Shortcut for adding a full size background image to an element
  • $imgUrl - Add url for background image

Example

.element {
    @include background-cover(img.png);
}
.element {
    background-image: image-url("img.png");
    background-repeat: no-repeat;
    background-size: cover;
}
